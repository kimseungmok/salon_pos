## Course 1: Getting Started with Fresha

### 1-1. Launch Your Workspace (인트로)
- Fresha Academy 소개 영상 (진행자: Sophie)
- 전체 코스 구조 안내: 계정 생성 → 비즈니스 설정 → 서비스 생성 → 팀 추가 → 시프트 스케줄링 → 예약
- **핵심 메시지**: "데스크톱과 모바일 모두 동일한 기능" → Fresha for Business 앱 병행 사용

### 1-2. Create an Account
- 이메일 또는 소셜 계정으로 가입
- 기본 정보 입력 + 비밀번호 생성
- **휴대폰 인증 코드** 확인
- 비즈니스명, 웹사이트, 주요 서비스 종류, 팀 규모 입력
- **지도 핀으로 정확한 위치** 설정 (온라인 예약 고객 길 찾기용)
- 기존 소프트웨어 전환 여부 + 유입 경로 입력

### 1-3. Add Business Details
- 워크스페이스 설정에서 비즈니스명, 세금, 언어 수정 가능
- SNS 링크·웹사이트 연결
- **다국어 지원**: 언어 설정 (일본어 포함)
- **세금 설정** (일본 적용 시: 소비세 10%/8% 이중 세율)
- 언제든 돌아와서 설정 수정 가능

### 1-4. Create Your First Service
- 카탈로그(Catalogue) → 서비스 메뉴 관리
- **카테고리** 먼저 생성 (예: 컷/컬러 분리)
- 서비스 기본 정보: 이름, 카테고리, 가격, 소요시간
- 고급 옵션: 가격 변형(Variants), 프로세싱 타임, 블록 타임
- 팀원별/리소스(세면대, 트롤리)별 서비스 제공 여부 설정
- 온라인 예약, 커미션 설정 연계 가능

### 1-5. Add Team Members
- **팀원 무제한 추가** + 초대 이메일 발송
- 기본 정보: 이름, 이메일, 집 주소, 비상연락처
- **프로필 사진** 설정
- 서비스 할당: 스타일리스트에게 서비스 연결, 리셉션은 생략
- **캘린더 예약 가능 여부** 설정
- **접근 권한 레벨** 설정 (역할별 권한 차별화)
- 초대 수락 → 팀원이 자신의 계정 생성

### 1-6. Schedule Shifts
- 팀 → 예약 가능한 시프트 설정
- **반복 스케줄**: 주기/시작일/종료일 설정
- 요일별 근무 시간 설정
- **하루에 여러 시프트** 추가 가능 (점심 휴식 등)
- **휴가 추가**: 정규 스케줄에 영향 없이 특정일 휴무
- 시프트 설정이 온라인 예약 가능 시간의 기준이 됨

### 1-7. Import Products
- 카탈로그 → 상품 관리
- **CSV 파일로 일괄 가져오기** (상품 목록 + 재고 수량)
- 템플릿 다운로드 → 기존 엑셀에서 복사 붙여넣기 → 업로드
- 컬럼 매핑 자동 조정
- 가져오기 전 미리보기로 확인
- 재고 현황이 자동 반영됨

### 1-8. Book Your First Appointment
- 캘린더 → 시간 슬롯 선택 → 서비스 추가 → 저장
- 팀원 컬럼 상단에 예약 가능 시프트 자동 표시
- 첫 예약 완료 = Fresha 기본 설정 완료

## Course 2.1: Schedule Appointments

### 핵심 기능들

**단일 예약 생성**
- 캘린더에서 빈 슬롯 탭 → 서비스 검색 또는 목록 선택
- **여러 서비스 동시 추가** 가능
- 날짜·시간 상단에서 직접 수정
- 서비스별 가격 조정·할인·팀원 변경·추가 시간 설정
- 기존 고객 선택 또는 **즉석 신규 고객 생성**
- 고객 선택 시 **알러지·멤버십·스태프 알림** 즉시 표시
- **내부 메모** 추가 (고객 비공개 — 팀 전용)

**예약 편집·취소·리스케줄**
- 드래그&드롭으로 시간 이동 ← **핵심 UX**
- 다른 날짜/팀원으로 이동: Quick Actions → 리스케줄
- 상태 업데이트: 확인/취소/노쇼 처리
- 서비스·가격·팀원 변경 → 즉시 캘린더 반영

**반복 예약 (Regular Clients)**
- 반복 주기 설정 (매주/2주/매월 등)
- 시리즈 전체 기간 설정
- 개별 예약만 수정 가능 (시리즈 전체 영향 없음)
- 고객에게는 **다음 예약 확인 이메일**만 발송

**그룹 예약**
- Add → Group Appointment 선택
- 기존 예약 그룹화 또는 신규로 시작
- 그룹 내 각 예약 개별 수정 가능
- **그룹 통합 체크아웃** (한번에 결제)
- 그룹 전체 취소 또는 그룹 해제 옵션
- 온라인 예약 시 고객들이 함께 예약 가능

## Course 2.2: Record a Sale (POS)

### 핵심 기능들

**세일 생성 흐름**
1. 완료된 예약 → 체크아웃 탭 → 카트 자동 구성
2. 카트에 추가 가능: 예약, 서비스, 상품, 멤버십, 기프트카드
3. **바코드 스캐너**로 상품 즉시 추가
4. **Quick Sale** 설정: 자주 쓰는 아이템 즉시 접근
5. Continue to Payment → 결제 수단 선택 → 완료
6. 완료 후 예약 상태 자동 `complete` 변경 + 감사 메시지 자동 발송
7. 영수증 즉시 출력 가능

**카트 커스텀**
- **전체 카트 할인**: Quick Actions → Cart Discount → 금액/% 선택
- **아이템별 할인**: 개별 아이템 탭 → 가격 수정 또는 기존 딜 선택
- 카트 할인 + 아이템 할인 **중첩 적용** 가능
- 수량 업데이트 및 담당 팀원 변경
- 세금·할인 자동 계산

**결제 수단 & 팁**
- 팁: 프리셋 % 옵션 또는 커스텀 금액
- 팁 팀원 분배 또는 **여러 팀원에게 분배**
- 고객 감사 메시지로 팁 추가 가능 (앱에서 고객이 직접)
- **현금 결제**: 받은 금액 입력 → **거스름돈 자동 계산**
- **분할 결제 (Split Payments)**: 카드 + 기프트카드 등 조합
- Fresha 결제 연동 시 **카드 온 파일** (등록 카드) 사용 가능

**환불 (Refund)**
- 완료된 세일 → 환불 옵션 선택
- **아이템 환불**: 특정 서비스/상품 선택 → 환불
- **금액 환불**: 커스텀 금액 + 사유 입력
- 시스템 자동 기록·업데이트
- 고객에게 환불 확인 발송
- **팀원 변경** (환불 없이): 세일 편집 → 팀원 재할당 (커미션 수정용)

## Course 2.3: Manage Clients

### 핵심 기능들

**고객 프로필 생성**
- 예약 중 또는 고객 목록에서 직접 추가
- 필수: 이름 / 권장: 전화번호 + 이메일
- 생일 등록 → 특별 혜택 발송 가능
- 주소 + 비상연락처 저장
- **알림 방식 설정**: SMS/이메일/양쪽
- **마케팅 수신 동의** 관리

**고객 프로필 스마트 표시**
- **신규 고객 라벨**: 첫 예약 완료 전까지 표시
- **노쇼 라벨**: 노쇼 이력 있으면 항상 표시 ← 중요
- **로열티 티어 + 포인트 잔액** 즉시 표시
- **활성 멤버십/패키지** 표시
- **알러지·패치테스트 결과** 기록
- **스태프 알림**: 모든 예약에서 팀에게만 보이는 경고

**고객 풀 프로필 구성**
- **Overview**: 최근 활동 스냅샷
- **Appointments**: 전체 예약 이력 + 예정 예약
- **Sales**: 모든 구매·환불 이력
- **Details**: 연락처·알림 설정 편집
- **Items**: 구매한 상품·서비스 목록 (세일 열지 않고 조회)
- **Documents**: 메모, 파일, 첨부(사진), 양식(동의서·문진표)
  - 컬러 레시피, Before/After 사진 → 노트에 이미지 추가
- **Wallet**: 등록 카드, 기프트카드, 보증금, 리워드
  - 리워드 직접 추가 가능 (무료 시술·할인)
- **Loyalty**: 포인트 잔액, 티어, 추천인 이력
  - 수동 포인트 추가/티어 변경 가능
- **Reviews**: 고객이 남긴 평점·리뷰 + 답변 가능
- **권한 제한**: 팀원이 연락처 보기 차단 가능 (예약·결제는 가능)

## Course 2.4: Master Your Calendar

### 핵심 기능들

**캘린더 뷰 옵션**
- **하루/3일/주간/월간** 4가지 뷰
- **빨간 라인**: 현재 시간 표시
- 날짜·위치·팀원·룸/장비별 필터
- **줌 레벨** 조정 (바쁜 날 더 많이 표시)
- 팀원 이름 탭 → 해당 팀원 캘린더 집중·시프트 관리

**색상 코딩 커스텀**
- **색상 기준 선택**: 팀원별 / 서비스 카테고리별 / 상태별
- 팀원 설정 + 서비스 카테고리에 개별 색상 지정
- **커스텀 예약 상태** 생성 (예: "내원 확인" 상태 추가)
  - 상태 색상 + 순서 설정
  - 기본 상태와 함께 워크플로우에 통합

**블록 타임 (시간 차단)**
- 타입: 청소·점심·미팅 등 프리셋 타입 생성
  - 이름 + 아이콘 + 기본 시간 + 유급/무급 설정
- 반복 블록 타임 설정 (매일 점심 등)
- 설명 추가 가능
- 블록된 시간은 온라인 예약에서 자동 제외

**웨이트리스트 (대기자 명단)**
- 예약이 꽉 찬 시간 → 고객이 대기 신청
- 빈 슬롯 생기면 **자동 알림 발송** → 고객 자동 예약
- 우선순위 설정: **선착순** or **고가 서비스 고객 우선**
- 또는 **전체 동시 알림** → 먼저 예약하는 고객 배정
- 인스토어에서 대기 추가 가능
- 대기 엔트리 편집·수동 예약·제거 가능

**휴업일 설정 (Closed Periods)**
- 날짜 범위 설정 → 전체 캘린더 + 온라인 예약 자동 차단
- 팀 시프트 오버라이드
- 고객 예약 화면에서 "다음 예약 가능일" 자동 안내
- 설명 추가 (팀 내부용)

## Course 2.5: Get the Most from Your Services

### 핵심 기능들

**서비스 생성 (상세)**
- 서비스명 + 카테고리 + 설명 (AI 자동 생성 지원)
- 사진 추가 (온라인 예약 고객 유인)
- **Starting From 가격**: 최종 가격이 상담 후 결정되는 서비스용
- **Variants (변형)**: 숏/미디/롱 등 옵션 — 서비스 여러개 만들 필요 없음
- **팀원별 다른 가격/소요시간**: 시니어/주니어 스타일리스트 차등
- 담당 팀원 지정 + 리소스(룸·장비) 연결

**Extra Time (추가 시간)**
- **Blocked Time (블록)**: 서비스 후 청소 등 — 고객 비공개, 팀원 다른 예약 불가
- **Processing Time (처리 중)**: 고객 있지만 팀원 자리 비울 수 있음 (염색 발색 대기)
  - 처리 중 팀원이 다른 예약 가능 ← **효율성 극대화**
  - 캘린더에서 부분 오픈 슬롯으로 표시
- **Extra Servicing Time**: 처리 후 팀원이 다시 고객에게 돌아오는 시간

**번들 (Bundle)**
- 여러 서비스 묶음 → 한번에 예약
- 예: 워시 + 컷 + 블로우 드라이 패키지
- **할인 적용** 가능 (묶음 구매 유도)
- 서비스 순서 지정
- **병렬(Parallel) 예약**: 동시에 여러 팀원이 시술하는 경우
- 온라인 예약 시 전체 내용·가격·소요시간 한눈에 표시

**서비스 Add-ons (c2_5_5)**
- 기본 서비스에 선택적 추가 서비스 붙이기
- 예: 기본 컷에 두피 트리트먼트 추가 옵션
- 고객이 온라인 예약 시 직접 추가 선택 가능

**Optimize Services List (c2_5_6)**
- 서비스 목록 정리·최적화 팁
- 명확한 이름·카테고리·설명의 중요성
- 메뉴 구조 간소화 → 예약률 증가

## Course 04: Accept Payments

### 핵심 기능들 (제목 기반 분석)

**Explore Payments (c4_1)**
- Fresha 결제 시스템 전체 소개
- 결제 수단 종류: 카드, 현금, 앱, 분할 결제
- 결제 통합의 비즈니스 이점

**Set Up Payments (c4_2)**
- Fresha Payments 계정 설정
- 카드 단말기 연결
- 결제 수수료 구조 안내

**Payment Options (c4_3)**
- 다양한 결제 수단 구성
- 카드 온 파일 (등록 카드) 설정
- 현금·카드·앱 결제 설정

**Paying with Tap to Pay (c4_4)**
- **Tap to Pay (비접촉 결제)**: 스마트폰으로 카드 단말기 없이 결제
- iPhone NFC를 이용한 결제 수락
- 일본 적용: Apple Pay / iD / QUICPay 연동 가능성

**Take Payments Using Card Terminals (c4_5)**
- 물리 카드 단말기 연결·설정
- 단말기 결제 흐름
- 일본 주요 단말기: Square, PAX, Verifone

**Payment Policies (c4_6)**

## 일본 살롱 현지화 핵심 적용사항

### Fresha에서 배운 일본 특화 포인트

1. **Processing Time 활용**: 일본 살롱은 퍼머·컬러 시술 시 대기 시간이 길어 이 기능이 매우 중요
   - `durationMin` + `bufferMin` → 실제 처리 시간 분리 표현 강화 필요

2. **노쇼 라벨 강화**: 일본은 예약 문화가 강하고 노쇼 시 손실이 큼
   - 고객 프로필에 노쇼 횟수·비율 표시 + 예약 시 경고

3. **문진표·동의서 (同意書)**: 일본 살롱에서 법적으로 중요
   - 패치테스트 동의서, 퍼머·컬러 시술 동의서 디지털화
   - Fresha: Documents 탭 내 폼 관리

4. **서비스 Variants**: 일본 살롱 가격 체계 (숏/미디/롱 by 길이, 등급 by 시술자)
   - 현재 MenuStaffPrices 테이블 활용 → UI로 구현 필요

5. **자동화 메시지 = LINE 발송**: 
   - 예약 확인: LINE 메시지
   - 리마인더: 전날 LINE
   - 감사 메시지: LINE (리뷰 요청 포함)

6. **번들 = 일본 살롱 코스 메뉴**: 
   - 예: カット+カラー+トリートメントセット
   - 세트 가격 할인으로 객단가 향상

7. **로열티 티어**: 일본 살롱 단골 관리 문화와 일치
   - ブロンズ/シルバー/ゴールド/プラチナ 4단계

8. **보증금 정책**: 일본 고급 살롱에서 도입 증가
   - 온라인 예약 시 신용카드 등록 필수 설정 옵션
