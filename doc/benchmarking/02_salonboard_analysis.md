# 살롱보드(SalonBoard) 기능 분석 — 기밀

> 분석일: 2026-06-06  
> 출처: 살롱보드 홈페이지 기능 소개 캡쳐 (사내 기밀)  
> 목적: 일본 살롱 POS 앱 기능 벤치마킹 및 현지화 적용

---

## 개요

살롱보드는 **일본 살롱 전용** 예약·고객관리·POS 통합 솔루션으로,  
일본 미용실 시장 점유율 최상위 SaaS 제품.  
특히 **일본어 UI, 일본 살롱 업무 흐름**에 최적화되어 있어  
우리 앱의 핵심 현지화 레퍼런스.

---

## 1. 24시간 네트 예약 (オンライン予約)

- **24시간 온라인 예약** 수락 — 고객이 웹/앱에서 직접 예약
- 예약 승인/거절 자동화
- **스마트폰 최적화** 예약 페이지 (고객용)
- 예약 완료 시 자동 확인 메시지 발송
- **예약 전날 리마인더** 자동 발송
- 외부 예약 사이트(ホットペッパービューティー 등)와 연동 가능

### 살롱 POS 적용 포인트
- 온라인 예약 링크 생성 → QR 코드로 매장에 부착
- 예약 → 당일 내원 시 POS 결제 자동 연동
- 리마인더: LINE 메시지 발송 (일본 고객 주요 채널)

---

## 2. 예약의 일원관리 (予約一元管理)

- **전 채널 예약 한곳 통합**: 전화예약, 온라인예약, 앱예약 → 단일 캘린더
- 스태프별 컬럼 뷰 캘린더 (우리 앱과 동일 구조)
- **드래그&드롭**으로 예약 이동
- 예약 상태: 미확인 → 확인 → 내원 → 완료 4단계
- **더블부킹 방지** 자동 체크
- 예약 변경/취소 자동 알림

### 살롱 POS 적용 포인트
- 현재 앱 캘린더에 **드래그 이동** 기능 추가 필요
- 예약 상태 4단계 흐름 확인 (현재 앱은 5단계)
- 더블부킹 방지 로직 강화

---

## 3. 顧客管理 (고객관리)

- **고객 카르테(カルテ)**: 시술 이력, 사용 제품, 퍼머·컬러 레시피 기록
- 사진 첨부 (Before/After)
- **알러지 정보**, 패치테스트 기록
- 고객별 **선호도, 주의사항** 메모
- **방문 주기 분석**: 마지막 방문일 기준 휴면 고객 파악
- **DM 발송 대상 필터**: 조건 필터 → 타겟 고객 추출

### 살롱 POS 적용 포인트
- 이미 카르테 기능 구현됨 (TreatmentRecords 테이블)
- **Before/After 사진 첨부** UI 강화 (현재 photos JSON으로만 저장)
- 알러지 정보 카르테 화면에 경고 배너로 강조

---

## 4. DM (다이렉트 메시지 / CRM)

- **조건 필터로 DM 발송**: 방문 후 N일 경과, 생일, 시술 종류 등
- **자동화 메시지**: 예약 확인, 리마인더, 감사 메시지, 생일 축하
- **LINE 연동**: 일본 살롱 고객 주요 소통 채널
- 메시지 템플릿 등록/관리
- 발송 이력 관리

### 살롱 POS 적용 포인트
- 현재 앱 `message_templates`, `campaigns`, `automation_rules` 테이블 존재
- **LINE 메시지 연동** 최우선 (일본에서 카카오 없음)
- 자동화 규칙: 방문 후 30일 → 자동 DM, 생일 전날 → 자동 DM
- 발송 채널: LINE > SMS > 이메일 순서 우선순위

---

## 5. レジ機能 (레지 / POS 기능)

- **간단 결제**: 메뉴 선택 → 금액 확인 → 결제
- 결제수단: 현금, 카드, QR결제, 포인트
- **영수증 발행**: 종이 + 전자영수증
- **일별 마감**: 매출 집계 → 레지 마감
- 당일 매출 확인 → 지점별 비교

### 살롱 POS 적용 포인트
- 일본 QR 결제: **PayPay, LINE Pay, au PAY** 주요 수단 (우리 앱 ic_card → PayPay로 레이블 변경)
- 레지 마감 = 우리 앱 CloseRegisterSheet

---

## 6. 集計・分析機能 (집계·분석)

- **매출 집계**: 일별/월별/연별 매출 차트
- **스태프별 매출**: 개인 실적 비교
- **메뉴별 인기도**: 시술 종류별 매출 순위
- **신규/재방문 비율**: 고객 유형별 분석
- **예약 충전율**: 예약 대비 실제 내원율
- 데이터 CSV 다운로드

### 살롱 POS 적용 포인트
- 리포트 화면: **스태프별 차트, 메뉴별 순위** 시각화 추가
- 예약 충전율 KPI 추가 → KpiTargets 테이블 활용
- CSV 내보내기 기능 (일본 살롱 경영자 엑셀 선호)

---

## 7. スマート支払い (스마트 결제)

- **iPad/스마트폰** 결제 지원
- 카드 단말기 연동 (일본 주요: Square, PAX, Verifone)
- **카운터리스 결제**: 고객이 자신의 스마트폰으로 직접 결제
- QR 결제 화면 고객에게 제시

### 살롱 POS 적용 포인트
- iPad Air 2 타겟 → Square 카드리더기 연동 검토
- QR결제 화면: 고객 제시용 전체화면 QR 뷰 구현

## 10. 일본 살롱 특화 시사점

1. **LINE 연동 최우선**: 일본 살롱 고객 소통은 LINE이 절대적
2. **PayPay/LINE Pay**: 일본 모바일 결제 시장 1-2위, 반드시 포함
3. **ホットペッパービューティー 연동**: 일본 최대 살롱 예약 플랫폼 (v2 로드맵)
4. **카르테 사진**: 일본 살롱은 시술 전후 사진 필수 문화
5. **경어체 UI**: 일본어 표현은 丁寧語(정중체) 기준
6. **영수증 = 領収書**: 일본 사업 영수증은 법적 서식 요건 존재
