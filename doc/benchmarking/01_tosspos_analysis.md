# 토스포스(TossPos) 기능 분석 — 기밀

> 분석일: 2026-06-06  
> 출처: 토스플레이스 공식 가이드 스크린샷 (사내 기밀)  
> 목적: 일본 살롱 POS 앱 기능 벤치마킹 및 현지화 적용

## 1. 결제 (Payment)

### 1-1. 기본 결제 흐름
- 메뉴 화면 → 상품 선택 → **결제 버튼** 탭 → 결제수단 선택 → 완료
- 결제수단: 카드, 현금, 간편결제(카카오페이·네이버페이 등), 포인트
- 결제 전 **할인 버튼**으로 즉시 할인 적용 가능
- 결제 화면에서 **금액 수동 입력** 지원 (시가 설정)

### 1-2. 분할 결제 (Split Payment)
- 하나의 주문을 **여러 결제수단으로 분할** 가능
- 예: 카드 3만원 + 현금 2만원 혼합 결제
- 금액 직접 입력 또는 비율 분할
- 분할 결제 취소/변경 지원
- **디지털 결제, 매뉴얼 결제, 선불(기프트카드) 결제** 각 타입별 분할 지원

### 1-3. 할인 (Discount)
- 선택한 메뉴 아이템별 개별 할인 적용 (아이템 할인)
- 전체 결제 금액 할인 (총 결제 금액 할인)
- **자주 쓰는 할인 저장** → 편집모드에서 즐겨찾기 등록
- **자동 할인**: 조건 설정(시간대·요일·메뉴) → 자동 적용
- 할인 방식: 금액(￦) / 비율(%) 선택

### 1-4. 결제수단 추가/커스텀
- 기본 제공 결제수단 외 **커스텀 결제수단 추가** 가능
- 결제수단 목록에서 이름 입력 → 추가 → **순서 드래그 편집**
- 불필요한 결제수단 삭제 가능
- 새로 추가한 결제수단이 결제화면에 즉시 반영

### 1-5. 결제내역 상세 조회
- 배달·포장·홀, 날짜별·직원별·메뉴별 필터 조회
- **카드 내역 찾기**: 카드사 프론트로 연동 → 거래 검색
- 상세 조회에서 영수증 재발행 가능
- 부분 취소/전체 취소 지원

### 1-6. 후불 매장 셀프결제 (Deferred Payment)
- **포스에서 후불결제 기능 ON**
- 고객이 테이블 QR 또는 키오스크로 직접 주문·계산
- 셀프 계산서 출력 설정 가능
- **(손님)계산서 출력, 계산하기/분할결제** 지원
- 프론트 직원 없이도 고객 자율 정산 가능

### 1-7. 매출 리포트 잠금
- 민감한 매출 리포트에 **PIN 잠금** 설정
- 설정 → 화면접근 권한 → 매출리포트 잠금 토글
- **비밀번호 재설정**: 잊어버렸을 경우 휴대폰 인증으로 재설정

## 2. 주문 (Order)

### 2-1. 선불형 상품 주문
- **선불형 매장** 전용 주문 흐름 (카페/편의점형)
- 주문 내역 추가 → 수량 조정 → **상단 바코드** 터치 시 주문 잠금
- 옵션 상품 적용 가능 (사이즈, 온도 등)
- 등록되지 않은 메뉴는 단발성 주문 가능
- **완료 버튼** → 영수증 상세 확인
- 주문 내역 삭제: 좌측 스와이프 or 편집 버튼
- 할인/세금 적용 후 주문 가능

### 2-2. 선불형 주문 현황
- **포스 주문 현황 페이지** (주문 대시보드)
- 뷰 종류: 접수기/진행중/완료 탭으로 분류
- **완료/취소 버튼** → 나중에 주문 리스트에서 재확인 가능
- **키오스크 주문** 연동 → 실시간 주문 현황 확인
- 주문 항목 클릭 → 상세 팝업 → 완료/취소 처리

## 3. 기본 설정 (Settings)

### 3-1. 즐겨찾는 메뉴·옵션 편집모드
- **카테고리 1줄(1행) 4개** 기준 배치
- 메뉴 이름 길면 자동 2줄 표시
- 즐겨찾는 메뉴 **표시/숨기기** 토글
- **상품 추가/삭제하기**: 카탈로그에서 선택
- **순서 편집**: 드래그로 위치 이동
- 셀/행 크기 편집 가능
- 즐겨찾는 옵션 **표시/숨기기/순서/삭제** 지원

### 3-2. 상품 대량 등록
- **포스 앱** 또는 **링크 로그인** 두 가지 방법
- 포스 앱: 설정 → 상품 → 상품 일괄 편집 → **엑셀 다운로드 후 내용 입력** → 업로드
- 수량이 늘어날수록 포스보다 **PC 링크 사용 권장**
- 주의사항: 상품등록 이미지, 수량의 너비 맞춤 필요, 특수 문자 사용 금지

### 3-3. 카테고리·옵션·상품 설정
- **카테고리** 추가/수정/삭제/순서 변경
- **카테고리 삭제 시**: 연결 상품은 전체 노출 전환
- **카테고리 숨기기** 지원 (임시 비노출)
- **카테고리 선택하기**: 체크박스 → 다중 선택
- **옵션 추가하기**: 옵션 그룹명 + 필수/선택 여부 + 옵션 항목 설정
- **옵션 삭제하기**: 연결 상품 해제 후 삭제
- **상품 추가하기**: 이름, 가격, 사진, 카테고리, 옵션 연결

### 3-4. 프린터·영수증·주문서 설정
- **영수증 커스텀 항목**: 가게명, 로고, 주소, 전화번호, 한 줄 메모
- 영수증 상단 이미지 or 텍스트 표시 선택
- **글꼴 크기**: 작게/보통/크게/가장 크게
- **할인 표시 방식**: 아이템별 or 총 합계에서 표시
- **프린터 속도 설정** (세이우/삼성 2개 모델별 가이드 제공)
- **영수증 쿠폰 번호**: 로또 번호 → 할인 비밀번호 설정
- **전자영수증 설정**: 카카오 or 문자 발송 (휴대폰 인증 필요)
- **종이영수증 추가 발행** 가능 (결제 후 재발행)

### 3-5. 업종 유형 변경
- **카페(선불)/음식점(선불)/음식점(후불)/도소매/서비스업** 5가지 유형
- 유형에 따라 UI 레이아웃 및 기능이 완전히 달라짐
- **서비스업** = 살롱 POS에 해당하는 유형
- **예약 기능**: 서비스업 유형에서만 활성화

## 4. 추가 세부 기능

### 4-1. 배달 앱 연동
- **배달의 민족 / 요기요 / 쿠팡이츠** 연동
- 포스 → 배달앱 연동 메뉴 → 각 플랫폼 연결
- 주문 자동 수신 → 포스에서 접수/거절
- **주문완료**: 포스에서 상태 업데이트 → 고객에게 알림
- 배달앱 별도 태블릿 없이 포스 단일기기로 통합 관리

### 4-2. 상품 바코드 스캔
- **설정 → 바코드 스캐너** 연결 (iOS/Mac 블루투스 기기)
- 바코드 스캔 → 해당 상품 자동 선택 → 수량 추가
- 바코드 미등록 상품 → 재고·판매관리로 등록 유도
- **키오스크 바코드 스캔**: 셀프 주문 중 바코드 스캔 기능

### 4-3. 시가 설정 (Variable Price)
- 가격 미정 상품에 **시가 설정** → 주문 시 가격 입력
- 메뉴에서 시가 아이콘 표시 → 탭 시 금액 입력 팝업
- **단발성 상품 주문** 에도 활용 가능

### 4-4. 시재관리 (Cash Drawer Management)
- **시재 설정**: 영업 시작 시 초기 현금 등록
- **돈 넣기**: 시재에 → 현재 잔액 확인 → 금액 입력
- **돈 빼기**: 시재에 → 금액 입력 → 메모 입력
- **돈통 정산하기**: 현금 금액별 수량 입력 → 자동 합산
- **현금 잔액 확인**: 포스 우측 상단 → 현금 잔액 조회
- **시재관리 설정**: 매출 종료 시 사용 여부/시재 다운로드 설정

### 4-5. 영수증 커스텀
- 로고, 주소, 전화번호, 사이트, 한 줄 메시지 설정
- **글꼴 크기** 4단계 조절
- **할인 정보 표시**: 아이템별 / 합계에서 표시 선택
- **영수증 쿠폰**: 로또 번호 + 비밀번호 설정
- 전자영수증: 카카오톡 or 문자 발송 연동
- 지원 모델: SEWOO SLK-T5100 / SAM4S GIANT-1005

### 4-6. 재고관리
- 포스 → 상품 → 재고관리 → **재고 수량 수정**
- 재고 기능 활성 시 판매될 때마다 재고 차감 + 부족 알림
- **재고 수량 빠른 수정**: 더하기(+) → 수량 입력 → 저장
- **재고 0이 되면**: 상품 자동 품절 표시
- **재고관리 히스토리**: 포스 → 재고관리 → 과거 이력 확인
  - 카테고리별, 날짜별, 변경 이유별 필터

### 4-7. 전자영수증 설정
- **카카오채널** 전자영수증 (비즈니스 계정 필요)
- **문자 SMS** 전자영수증 (유료 서비스)
- 결제 후 → 고객 휴대폰 번호 입력 → 전자영수증 발송
- **종이 영수증 추가 발행** 토글로 병행 출력

### 4-8. 전화주문 (CID 연동)
- **KT/LGU+ 통신사 연동**으로 발신번호 자동 인식
- 전화 오면 → 고객 정보 자동 팝업 (이름, 방문이력, 주문내역)
- 등록 안 된 고객이면 → 신규 고객 등록 유도
- 전화 기록 30일 내 재조회 가능
- **주의**: 동일 서비스가 가입된 경우 통신사 변경 불가

### 4-9. 포스 고객 호출 방식
- **진동벨**: 고객에게 번호표 배부 → 포스에서 번호 호출
  - 화면에 **진동벨** 아이콘 탭 → 번호 입력 → 호출
  - 최대 100번까지 동시 관리
- **알림톡(카카오)**: 고객 휴대폰으로 카카오 알림톡 발송
- **프론트 패널 알림톡**: 주문서 QR 스캔 → 고객 폰으로 발송
- **포스 주문 현황 페이지 알림톡 호출**: 현황 화면에서 알림톡 호출 버튼
- **키오스크 알림톡 설정**: 키오스크에서 주문 완료 시 자동 발송

## 5. 예약 관리·예약 결제

- **예약 결제**: 예약 시 선결제(보증금) 기능
- 예약 시 결제 방법 선택 → 온라인 링크 or 현장 결제
- 예약 상태: 대기 → 확인 → 완료 흐름
- 예약 + POS 결제 연동: 예약에서 바로 결제 화면 진입 가능

| 패턴 | 설명 |
|------|------|

| 탭바 기반 | 주문현황 → 접수중/진행중/완료 탭 분리 |

## 7. 살롱 POS 적용 포인트

### 즉시 적용 가능
- [ ] **분할결제** UI: 결제화면에서 수단별 금액 분리 입력
- [ ] **즐겨찾기 메뉴 편집모드**: 자주 쓰는 시술 상단 고정
- [ ] **아이템별 할인 + 전체 할인** 두 단계 할인 구조
- [ ] **시재관리**: 돈 넣기/빼기/정산 흐름
- [ ] **영수증 커스텀**: 살롱명 로고 + 한 줄 메모
- [ ] **재고관리 히스토리**: 제품 재고 변동 이력 조회
- [ ] **매출리포트 PIN 잠금**: 민감 데이터 보호

### 일본 살롱 특화 적용
- [ ] **전자영수증**: 일본은 LINE 연동으로 대체 (카카오 없음)
- [ ] **전화주문 → 예약전화 자동 고객 인식**: CID 대신 IP 전화 연동 검토
- [ ] **시가 설정**: 살롱에서 컨설팅 후 가격 결정하는 시술에 활용
- [ ] **고객 호출**: 진동벨 연동 or 스마트폰 알림으로 대기 호출
