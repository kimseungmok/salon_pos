## 경쟁사별 최강점 (우리가 반드시 흡수해야 할 것)

### 토스포스 강점
- **시재관리 완성도**: 돈넣기/빼기/정산 전용 흐름
- **즐겨찾기 편집모드**: 드래그 순서 변경, 표시/숨기기
- **분할결제 UI**: 수단별 금액 분리 입력, 취소/변경
- **영수증 커스텀**: 로고·글꼴·쿠폰·전자영수증 완벽 지원
- **자동 할인**: 조건(시간대·요일)별 자동 적용

### 살롱보드 강점 (일본 시장 기준)
- **LINE 연동**: 예약 확인·리마인더·DM 모두 LINE
- **ホットペッパービューティー 연동**: 일본 최대 예약 플랫폼
- **완전 일본어 UI + 존댓말**: 현지화 레퍼런스
- **고객 카르테 사진**: Before/After 필수 문화 반영
- **자동 DM**: 방문 후 N일, 생일 자동 발송

### Fresha 강점
- **Processing Time**: 퍼머·컬러 시술 중 다른 고객 동시 예약 → 매출 극대화
- **번들(セット)**: 묶음 예약 + 할인 → 객단가 향상
- **웨이트리스트 자동화**: 취소 슬롯 자동 채우기
- **로열티 티어**: 단골 등급 관리 + 티어별 혜택
- **고객 문서**: 동의서·문진표 디지털 관리
- **서비스 Variants**: 길이·등급별 가격 차등

## 개발 재개 권장 순서

1. **결제 화면** 강화: 분할결제 + 아이템 할인 + 환불 UI
2. **고객 프로필** 강화: 노쇼 라벨 + 스태프 알림 + 사진 첨부
3. **리포트 화면** 구현: 차트 + 스태프별 + 메뉴별
4. **로열티/기프트카드/멤버십** UI 구현
5. **설정** 강화: 영수증 커스텀 + 시재관리 + 즐겨찾기
6. **예약 캘린더** 강화: Processing Time 시각화 + 드래그
7. **재고 관리** UI 구현
8. **LINE 연동** 착수 (v2)
