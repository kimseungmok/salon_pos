## 메뉴 카드 상세

```
┌──────────────────┐
│  [썸네일 이미지]  │  ← 72×72pt 이미지 또는 색상 블록
│                  │
│  ✂ 컷트 (숏)     │  ← 메뉴명
│  ¥ 4,000        │  ← 가격 (굵게, 큰 폰트)
│  60분            │  ← 소요시간 (회색 작은 텍스트)
└──────────────────┘
  카드 크기: 136×120pt
  탭 시: 주문서에 즉시 추가 + 살짝 눌리는 애니메이션
  길게 탭: 옵션 선택 바텀시트 열림
```
