- 빈 칸 탭 → 새 예약 생성
- 예약 블록 탭 → 예약 상세/편집
- 예약 블록 길게 탭 → 드래그로 시간 이동
- 예약 블록 우측 하단 드래그 → 시간 연장
- 스태프 헤더 탭 → 해당 스태프만 필터
