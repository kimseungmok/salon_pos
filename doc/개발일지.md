- `AlertDialog`로 교체 테스트 → 정상 → `showDialog` context 자체는 문제 없음
- 원인: `ConsumerStatefulWidget with SingleTickerProviderStateMixin` 조합이 `showDialog` overlay context에서 충돌
- 시도1: `ConsumerStatefulWidget` → `StatefulWidget` → 동일 증상
- 시도2: `SingleTickerProviderStateMixin` 제거 + `DefaultTabController` → 동일 증상

**최종 해결:**
- `_showDetail` 메서드를 `async`로 변경, 데이터 먼저 로드
- `_CustomerDetailDialog`를 `StatelessWidget`으로 완전 재작성 (State 없음, 데이터를 파라미터로 전달)

**사용자 불만:**
> "40분 내내 빌드를 하고 있냐 제정신이냐"
→ 빌드 반복이 너무 많아서 40분 소요. 이후 개선.
