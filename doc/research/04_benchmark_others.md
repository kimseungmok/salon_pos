# 기타 살롱 POS 벤치마킹

---

## Vagaro (미국 1위 살롱 SW)
> https://www.vagaro.com

### 핵심 특징
- 살롱·스파·메디컬 스파·바버 전문
- **MySite**: 살롱 전용 예약 웹사이트 자동 생성
- **Vagaro Capital**: 매출 기반 소상공인 대출
- **Branded App**: 살롱 브랜드 전용 앱 생성

### 참고 기능
| 기능 | 인사이트 |
|------|---------|
| Pay Later | 나중에 결제 (후불) |
| Forms | 고객 인테이크 양식 디지털화 |
| Reports | 상세 보고서 |
| Timeclock | 출퇴근 타임스탬프 |
| Commission | 스태프별 커미션 자동 계산 |
| Booth Rental | 의자 임대 관리 (일본 살롱 면도사 임대) |

## Phorest (아일랜드, 살롱 특화)
> https://www.phorest.com

### 핵심 특징
- 살롱 전용 CRM 최강
- **Smart Requests**: 고객 메모에서 AI가 선호도 자동 파악
- **Treatcard**: 포인트 카드 (앱 없이 SMS로 운용)
- **Client ReConnect**: 이탈 고객 자동 재방문 유도

### 참고 기능
| 기능 | 인사이트 |
|------|---------|
| Before & After Photos | 시술 전·후 사진 고객 카르테에 저장 |
| Patch Test Records | 패치 테스트 기록·알람 |
| Consultation Forms | 디지털 상담 양식 |
| Room/Equipment Management | 룸·설비 예약 관리 |
| Stock Management | 재고 최저 재고 알람 |

## Mindbody (피트니스·뷰티 통합)
> https://www.mindbodyonline.com

### 핵심 특징
- 클래스·멤버십 중심 (헬스장, 요가, 필라테스)
- **ClassPass** 연동 (외부 플랫폼 고객 유입)
- 회원권·정기권 관리 강함

### 참고 기능
| 기능 | 인사이트 |
|------|---------|
| Package/Series | 패키지·회수권 (예: 10회권) |
| Auto-pay | 멤버십 자동 결제 |
| Waitlist | 대기 명단 자동 알림 |
| Analytics 2.0 | 고급 분석 대시보드 |

## Booksy (바버·네일 특화)
> https://booksy.com

### 핵심 특징
- 심플한 모바일 중심 UI
- 고객이 직접 예약 (마켓플레이스 방식)
- 바버숍·네일숍에서 인기

### 참고 기능
| 기능 | 인사이트 |
|------|---------|
| Boost | 마켓플레이스 내 광고 부스트 |
| Message Blast | 일괄 메시지 발송 |
| No-show Protection | 노쇼 방지 (카드 보증금) |

## Square Appointments (글로벌)
> https://squareup.com/appointments

### 핵심 특징
- 무료 티어 존재 (개인 사업자)
- Square 결제 단말기와 완벽 연동
- **Resource Management**: 의자·룸 단위 예약

### 참고 기능
| 기능 | 인사이트 |
|------|---------|
| Resource Booking | 의자·룸 단위 예약 블록 |
| Prepayment & Deposits | 선불·보증금 |
| Multiple Locations | 다점포 관리 |
| Team Plus | 스태프별 권한 세분화 |
