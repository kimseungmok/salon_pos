# SalonBoard 벤치마킹 분석

> 출처: https://salonboard.com/function/
> Recruit Co., Ltd. 운영 | 일본 최대 살롱 예약·관리 시스템

---

## 기능 일람

### 예약관리
| 기능 | 설명 |
|------|------|
| 예약·스케줄 관리 | 살롱 전체 및 담당 스태프별 예약·예정 확인·변경·취소. 키레이살롱(네일·속눈썹·릴랙스·에스테) 전용은 설비축 스케줄도 확인 가능 |
| 예정·시프트 설정 | 시프트·휴식 등 예정 등록 |
| 예약 데이터 출력 | 예약 정보 CSV 등 데이터 출력 |

### 고객관리
| 기능 | 설명 |
|------|------|
| 고객 정보 관리 | 고객 정보 등록·확인·수정, 예약 이력 확인, 배신 메시지 확인 |
| 고객 정보 일괄 등록·변경·삭제 | 외부에서 관리하는 고객 정보 일괄 등록·변경·삭제 |

### 메시지 배신
| 기능 | 설명 |
|------|------|
| 일제 배신 | 고객 특정 개별 배신, 조건 지정 일제 배신 |
| 자동 배신 | 타이밍·조건 설정으로 대상 고객에게 자동 메일 배신 |
| 살롱 다이렉트 | 자사 홈페이지에 네트 예약 기능 설치 URL 발행 |

### 레지(POS)
| 기능 | 설명 |
|------|------|
| 회계 기능 | 회계 처리·레지 마감 |
| 매출 관리 | 일보·월보 등 매출 보고·명세 출력 |

### 집계·분석
| 기능 | 설명 |
|------|------|
| 집계 기능 | 기간별·메뉴별·스태프별 매출 실시간 확인 |
| 분석 기능 | 전년 대비, 리피트 상황, 내점 사이클, 우량 고객 분석 |

### 결제
| 기능 | 설명 |
|------|------|
| 스마트 지불 | 고객이 카드 정보 사전 등록 후 예약 → 내점 시 온라인 결제, 현지 회계 불필요 |

### 掲載관리 (HOT PEPPER Beauty 연동)
| 기능 | 설명 |
|------|------|
| 掲載정보 편집 | HPB 살롱 정보·스태프·포토갤러리·메뉴·쿠폰 편집 |
| 블로그 기능 | HPB 블로그 투고 |
| 구치코미 반신 기능 | HPB 네트 예약 고객 구치코미(리뷰) 반신 |

## 핵심 특이사항

1. **HPB 종속성**: HOT PEPPER Beauty 게재가 이용 필수 조건 → 우리 앱 차별점: 독립 운영 가능
2. **완전 무료**: 기능 전부 무료, 단 HPB 게재 필요
3. **iPad 지원**: PC/iPad 등 단말 + 상시 인터넷 연결 환경 필요
4. **헤어살롱 / 키레이살롱** 2가지 업종 구분 지원

## 우리 앱에 반영할 인사이트

- 스태프별 예약 스케줄 뷰 (타임라인 형식)
- 설비축 뷰 (예: 의자·부스 단위 스케줄)
- 우량 고객 분석 (리피트율·내점 사이클)
- 자동 메시지 배신 (리마인더·생일·감사)
- 一斉배신 기능 (조건 필터 고객 대상)
