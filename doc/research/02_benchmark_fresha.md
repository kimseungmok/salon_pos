## 전체 기능 체계

### 1. 일정 예약 (Scheduling)
| 기능 | 설명 |
|------|------|
| 캘린더 | 모든 디바이스 지원 통합 캘린더, 팀원·리소스 실시간 현황 |
| 예약 | 내장 양식·선불 결제로 보안 예약 생성 |
| 결제 | 카드·기타 결제수단으로 예약 결제 |
| 서비스 | 일정에서 직접 서비스 추가·맞춤 설정 |
| 리소스 | 서비스·예약에 리소스 생성·할당·자동 할당 |
| 자동 메시지 | 맞춤형 자동 알림으로 예약 이행률 극대화 |
| 대기 명단 | 라스트 미닛 예약 공백 자동 관리 |
| 그룹 예약 | 여러 게스트와 함께 예약 |
| 고객 | 예약에서 양식·패치 테스트·문서·메모 직접 관리 |
| 결제 정책 | 카드 확인·선불로 노쇼·취소 최소화 |

### 2. 고객 관리 (CRM)
| 기능 | 설명 |
|------|------|
| 고객 프로필 | 알레르기·패치 테스트·양식·메모·예약 이력 전부 추적 |
| 고객 지갑 | 잔액·기프트카드·멤버십·패키지·저장 카드 통합 관리 |
| 스마트 알림 | 자동화 SMS·이메일·앱 알림 |
| 사용자 지정 양식 | 예약 전 디지털 양식으로 필수 정보 미리 수집 |
| 고급 메모 | 시술 기록·사이즈 차트 보관 |
| 양방향 커뮤니케이션 | 고객과 직접 메시지 주고받기 (받은 편지함 통합) |
| 로열티 프로그램 | 포인트·등급·인센티브로 반복 방문 보상 |

### 3. 팀 관리
| 기능 | 설명 |
|------|------|
| 개인 계정 | 팀원별 개별 로그인·작업 공간 |
| 사용자 정의 권한 | 제품 판매~예약 확인까지 팀원별 접근 권한 |
| 교대 근무 일정 | 유연한 근무 일정표 |

| 급여 정산 | 자동 계산·기간별 급여 요약 |

## 우리 앱에 반영할 인사이트

- **고객 지갑 개념**: 잔액·포인트·쿠폰·기프트카드를 고객별 하나의 지갑으로 관리
- **패치 테스트·알레르기 기록**: 살롱 특화 안전 기록
- **스마트 추천 (업셀링)**: 결제 단계에서 관련 상품 자동 제안
- **시술 기록 카르테**: 고급 메모 → 전회 시술 내용, 색상 레시피 등
- **팀 목표 KPI**: 스태프별 매출 목표 설정·달성률 시각화
- **비혼잡 시간대 할인**: 한가한 시간대 자동 할인으로 예약 유도
