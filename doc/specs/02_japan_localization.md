# 일본 현지화 명세

---

## 필수 법령 대응

### 適格請求書 (인보이스 제도, 2023.10~ 시행)
- 발행자 등록번호 표시 (T + 13자리)
- 세율별 구분 기재 (8% 경감세율 / 10% 표준세율)
- 세율별 소비세 합계 금액 표시
- 예: "10%対象 ¥13,200 (うち消費税 ¥1,200)"

### 영수증 필수 기재 항목
```
[매장명]
[주소·전화번호]
[등록번호: T0000000000000]

[날짜·시간]
[영수증 번호]

[상품·서비스명]  [금액]
...
[소계]
[割引]
[10%対象]        [¥XX,XXX]
 内消費税10%      [¥X,XXX]
[合計]           [¥XX,XXX]

[결제 수단 내역]
[담당 스태프명]

[감사 메시지]
```

## 날짜·시간 표기

```dart
// 西暦 (서력)
2025年1月13日（月）

// 和暦 (일본 원호)
令和7年1月13日

// 시간
10:30 ~ 11:30

// 영수증
2025年01月13日 10:30
```

---

## 금액 표기

```
¥1,200          // 1,000 이상 3자리 구분
¥13,200         // 정수 (엔화 소수점 없음)
¥1,320,000      // 백만 단위도 3자리 콤마
```

## 입력 최적화

- 이름 입력: 한자 → 자동으로 후리가나 변환 추천
- 전화번호: 090-XXXX-XXXX 형식 자동 포맷
- 우편번호: 〒XXX-XXXX → 주소 자동 완성
- 금액 입력: 숫자 패드 (소수점 없음, 엔화 전용)
