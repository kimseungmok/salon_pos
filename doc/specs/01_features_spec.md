# 기능 명세서

> 버전: v0.1.0 기준 | 연매출 40억 살롱 대응 규모

---

## Phase 1 — 예약 + 고객 + 기본 POS (v0.1.0)

### F-001: 고객 등록·관리
- 이름(필수), 후리가나, 전화번호, 이메일, 성별, 생년월일
- 알레르기 태그 (복수 선택)
- 패치 테스트 날짜·결과 기록
- 일반 메모 / 스태프 전용 메모
- 고객 번호 자동 발행
- 소프트 삭제 (데이터 보존)
- 검색: 이름·후리가나·전화번호 빠른 검색

### F-002: 시술 기록 (카르테)
- 고객별 방문 이력 타임라인
- 헤어 길이, 색상 레시피, 파마 약제 상세
- 시술 전·후 사진 첨부 (로컬 저장)
- 다음 방문 주의사항 메모
- 방문 횟수·누적 매출 자동 집계

### F-003: 예약 관리
- 주간·일간 캘린더 뷰 (스태프 축)
- 예약 생성: 고객 + 스태프 + 메뉴(복수) + 시간
- 상태 관리: 대기→확정→진행중→완료 / 취소·노쇼
- 드래그 이동 (시간 변경)
- 예약 중복 방지 (같은 스태프 동시간대)
- Draft 자동 저장 (중단 시 복구)

### F-004: 메뉴 관리
- 카테고리별 메뉴 그룹 관리
- 메뉴: 이름(한/일), 가격, 소요시간, 세금 유형
- 옵션 그룹 (필수/선택, 복수 선택 가능)
- 즐겨찾기 메뉴 (빠른 접근)
- 대량 등록 (CSV 가져오기)

### F-005: POS 결제
- 메뉴 그리드 선택 → 주문서 자동 계산
- 소비세 10% 자동 적용 (내세금 표시)
- 할인: 금액·비율·쿠폰
- 결제 수단: 현금·카드 (기타는 Phase 8)
- 분할 결제 (예: 카드 + 현금)
- 팁 추가
- 영수증 프린터 출력 / 전자영수증
- Draft 자동 저장

### F-006: 기본 리포트
- 일별 매출 요약 (매출·건수·평균 객단가)
- 결제 수단별 집계
- 스태프별 매출

## Phase 2 — 스태프·시프트·재고 (v0.2.0)

### F-007: 스태프 관리
- 스태프 등록: 이름·역할·PIN·커미션율·캘린더 색상
- 역할별 권한 (오너·매니저·스타일리스트·어시스턴트)
- 시프트 등록: 날짜별 근무시간·휴식·휴일
- 월간 시프트 캘린더 뷰

### F-008: 커미션 계산
- 스태프별·메뉴별 커미션율 설정
- 기간별 커미션 집계 리포트
- 급여 명세 출력

### F-009: 재고 관리
- 상품 등록: 이름·바코드·원가·판매가·재고
- 결제 시 재고 자동 차감
- 재고 경고 알림 (설정 기준 이하)
- 수동 재고 조정
